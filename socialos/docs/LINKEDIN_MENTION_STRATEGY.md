# LinkedIn Mention Strategy

## Goal
Improve LinkedIn publishing without pretending full mention automation is always reliable.

## Phase 1
Suggested mentions only:
- infer mention candidates from linked people on the event
- prioritize contacts that already have LinkedIn identity data
- present them in the LinkedIn draft and handoff flow

## Phase 2
Assisted browser fill:
- during assisted publish, attempt to type or stage `@Name`
- let the user confirm the platform-side match

## Phase 3
Deeper automation later:
- only after identity quality and assisted fill reliability are high enough
- never as the first trust boundary

## Default Principle
Mention help should make handoff easier, not increase false positives or hide uncertainty.
